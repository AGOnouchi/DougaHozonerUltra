使い方

１．ffmpegを予めインストールしてください。
　　同梱されている「ffmpegインストール方法」を読んでください。

２．動画保存erUltraV〇.exeを起動します

３．保存したい動画のURLを入れます

４．ダウンロードを押して保存先を指定して保存します

５．おわり


年齢制限付きやメンバー限定の動画保存について
firefoxから自動取得するか、Get cookie.txtなどのソフトを用いてYoutube等のトップページからCookieを取得してください。（18歳以上やメンバー登録してあるアカウントでログインしておいてください）
ダウンローダーの画面に「Cookieファイルを選択」というボタンがあるのでそこでCookie.txtを選択してください。
この状態で年齢制限やメンバー限定の動画のURLを入れると保存できます。

ダウンロード時間について
サーバー上ではなく、各々のパソコン上で動作するのでダウンロード＆結合時間はお使いのPCのスペックによって変わります。


Abemaなどのサブスクサイトについて
yt-dlpは対応していますが、動画が暗号化されているためダウンロードできません。
ただし下記は行けたので対応してます
・TVer


Q.なぜffmpeg使ってるの？
A.ffmpegは動画と音声のダウンロードを別に行うのでそれぞれに処理が集中し、画質や音質が上がります。
本ソフトが8Kに対応しているのもそのおかげです。
