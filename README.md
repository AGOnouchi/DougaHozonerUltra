Q.年齢制限付きやメンバー限定の動画保存について

A.firefoxから自動取得するか、Get cookie.txtなどのソフトを用いてYouTube等のトップページからCookieを取得してください。

　（18歳以上やメンバー登録してあるアカウントでログインしておいてください）

 
　ダウンローダーの画面に「Cookieファイルを選択」というボタンがあるのでそこでCookie.txtを選択してください。
 
　この状態で年齢制限やメンバー限定の動画のURLを入れると保存できます。


Q.yt-dlpが対応してるサブスクサイトの保存について

A.無料で見れるものはダウンロード可能です。

　検証はしていないのですが有料サイトはログインしてあるCookieがあれば保存できると思います。
